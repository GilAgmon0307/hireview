The Solution

A platform built to evaluate how engineers actually work

HireView gives candidates a real coding environment with AI assistance -- then captures and scores their entire workflow.

For Recruiters

  • Create assignments or use AI-generated tasks tailored to role and stack
  • 11-criteria AI evaluation with scores from 0 to 100
  • Full session replay: timeline scrubber, file state at any point, chat log
  • Candidate comparison and percentile ranking across assignments
  • Suggested follow-up interview questions from session data
  • Group messaging with candidate and interviewer per assignment

For Candidates

  • Browser-based IDE (Monaco / VS Code engine) with multi-file support
  • Built-in AI chat: Ask, Plan, and Agent modes
  • Multi-language code execution: Python, TypeScript, JavaScript, C
  • Auto-save, keyboard shortcuts (Ctrl+S, Ctrl+Enter), file explorer
  • Session timer with countdown, progress bar, and time warnings
  • Or work locally with Claude Code and upload results

For Interviewers

  • Assigned per-candidate by the recruiter for access control
  • Watch full session replay after the candidate completes
  • Submit independent evaluation (1-5 rating + written notes)
  • Access group messaging thread with recruiter and candidate
  • Create and manage assignments, browse benchmark library
  • See AI evaluation alongside your own for calibration
